Техническое задание на проект: «Сервис анализа влияния атмосферного давления, температуры, влажности и фаз луны на качество сна»

1. Цель проекта: Создание веб-сервиса, который помогает людям выявить зависимость между погодными условиями (атмосферное давление, температура, влажность, фаза Луны) и качеством их сна. Пользователь ежедневно вносит данные о сне, а система автоматически дополняет запись метеоданными и строит наглядную аналитику.

2. Роли пользователей:

       • Гость: просмотр главной страницы с общим описанием сервиса, регистрация и вход в систему

       • Пользователь: внесение данных о сне (время засыпания/пробуждения, субъективная оценка о качестве сна), просмотр личного дневника сна, редактирование и удаление записей, просмотр статистики.

       • Администратор: управление пользователями (блокировка, удаление), просмотр всех записей всех пользователей, добавление/редактирование городов (для API-запросов), настройка параметров API (ключ, URL), просмотр логов интеграции с API. |

3. Модели данных (сущности):

       1. User profile (Профиль пользователя): Данные о пользователе.
          - Поля: name, city, birth_date, mail.
       2. Sleep diary (Дневник сна): Записи данных о времени и качестве сна.
          - Поля: user (связь ForeignKey), data, bedtime, wake_time, sleep_quality.
       3. Statistics (Статистика): Анализ влияния метеоданных на качество сна.
          - Поля: user (связь ForeignKey), recording date, sleep_record, pressure, humidity, moon_phase.

4. Ключевой функционал:

       1. Регистрация и настройка профиля: Пользователь региструется на сайте, указывает имя, дату рождения, город проживания, почту.

       2. Внесение данных о сне: Пользователь добавляет данные о сне, может их редактрировать или удалять.

       3. Просмотр статистики: На странице отображаются метеоданные и их влияние на качество сна пользователя.

5. Внешние интеграции / Аналитика:

       1. Интеграция: Использование внешнего API Яндекс Погоды для получения актуальных метеоданных (атмосферное давление, температура, влажность, фаза Луны) по географическим координатам города пользователя. API возвращает данные в формате JSON, которые автоматически сохраняются в базу данных и связываются с каждой записью о сне.

       2. Аналитика: Использование библиотеки Pandas для расчета статистических показателей по записям пользователя: средняя оценка качества сна, средняя продолжительность сна, день с максимальной и минимальной оценкой, коэффициент корреляции между атмосферным давлением и качеством сна. Данные выводятся в виде структурированных таблиц на странице аналитики без построения графиков (информация представлена в текстовом и табличном формате).
